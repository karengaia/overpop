Publishing Sequence
A page is displayed two different ways: dynamically and statically. Generally the public sees the static page and someone working as a volunteer sees the dynamic page, which is generated on the fly so you can see your results immediately.

An example of a dynamic URL is
http://www.population-awareness.net/cgi-bin/cgiwrap/popaware/article.pl?display_section%%%India_intro%
One drawback to dynamic pages is that images are difficult to display and we usually do not attempt to display them.

If you want to see the static page, you have to go to the location of that page - indirectly by going to the front page and navigating to the page using the indexes - or directly if you know the URL, like www.population-awareness.net/India.html

Once a webpage has been modified by a volunteer, it goes to the admin for approval, and is then FTP'd as a static page to the public area (in otherwords, it is 'published'). When a section editor makes changes, the page may advance to the public area immediately.
